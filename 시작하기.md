# kyt04100.github.io 배포 가이드

YT 정보로 커스터마이징된 al-folio 홈페이지입니다. 아래 순서대로 하면 됩니다.

## 1. GitHub 저장소 만들기
1. https://github.com/new 에서 저장소 이름을 정확히 **`kyt04100.github.io`** 로 생성 (Public, README 없이 빈 저장소)

## 2. 코드 업로드
이 폴더에서 터미널 열고:
```bash
git init
git add .
git commit -m "Initial personal website"
git branch -M main
git remote add origin https://github.com/kyt04100/kyt04100.github.io.git
git push -u origin main
```

## 3. GitHub Pages 활성화
1. 저장소 → **Settings → Pages**
2. Source: **Deploy from a branch**, Branch: **`gh-pages`** / root 선택
   - `gh-pages` 브랜치는 push 후 GitHub Actions(deploy.yml)가 자동 생성합니다. Actions 탭에서 빌드가 끝난 뒤(1~2분) 선택하세요.
3. 몇 분 뒤 https://kyt04100.github.io 접속 확인
